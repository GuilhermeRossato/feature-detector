How large are your features?

The smallest size necessary to detect what you're looking for. The size of the "eye" of the neural network. This is directly correlated to the input size of the solution.

How spaced should your inputs be?

How insensitive the network is for individual pixels. Spacing decreases the resolution of the input, significantly increasing the efficiency but making it less precise at detecting feature borders.

What is the shape of your features?

Focuses the network input, removing unnecessary inputs. Diamond shape is good for features with many shapes. Circle is especially good for round features. Square is foolproof, but slower.

How many channels are required?

Choose which colors may influence the detection of your features. Grayscale mode is only able to detect features by light intensity. HSL mode performs internal conversion from the RGB input.

How complex is your feature?

Each layer allows the network to learn an extra linearly separated aspect of the input to detect features. A 3-layer network can learn deeper mathematical relationships from the input when compared to a 1-layer network.

How many neurons in each internal layer

Affects the number of connections between the layers. Too many neurons may cause overfitting, where the network performs well on training data but is bad at generalizing on new images.

The internal activation function between layers

The function determines when a collection of neurons activate the next neurons (or output) by normalizing and shaping its aggregated results.

The number of full dataset passes

How many times the network will train with the full dataset. Depends on the complexity of your features. Large values will most definitely cause overfitting.
